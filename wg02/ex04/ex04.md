# Exercise 04 - SVG 라인 애니메이션

| Exercise 04              
| ------------------------ 
| 제출 디렉토리 : `ex04/`  
| 제출할 파일 : `ex04.html` and `ex04.css` 

`ex04.html`과 `ex04.css`를 직접 작성해야 합니다.

- `check.svg` 를 필수로 사용해 원형 버튼을 생성합니다.
- 기본 상태는 회색 라인의 체크 버튼입니다.
- 버튼을 클릭하면 녹색 체크 라인이 먼저 그려지고 원이 나중에 그려집니다. 
- 1회 재생시간은 3초 이하입니다.
- stroke 옵션을 잘 생각해보세요.
